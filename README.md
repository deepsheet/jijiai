# 急急话术AI

销售通话实时 AI 话术辅助桌面工具

## 项目简介

急急话术AI是一款运行在电脑桌面的智能辅助助手，像一位隐形的"金牌导师"，全程旁听您的腾讯会议/电话通话，并实时在屏幕悬浮窗推送最佳回复建议。

**核心价值**：帮助销售团队提升转化率、降低培训成本、避免合规风险。

## 快速了解

### 核心功能
- 🎯 **实时话术导航**：客户话音刚落，屏幕即刻弹出推荐话术
- 🧠 **企业知识库即时调用**：从上传文档中自动提取准确答案
- ⚠️ **风险合规预警**：实时监测并警示违规承诺
- 📊 **智能通话小结**：通话结束后自动生成结构化摘要

### 技术特点
- 🔒 **本地化部署**：敏感数据不出电脑，隐私安全有保障
- ⚡ **毫秒级响应**：端到端延迟 < 1.5 秒
- 💰 **超低成本**：99元/月起，无需购买昂贵SaaS系统
- 🌐 **跨平台支持**：Windows 10/11 (优先), macOS 12+

### 商业模式
- **基础版**：99元/人/月 (实时话术 + 100MB知识库)
- **专业版**：199元/人/月 (无限知识库 + 云端备份 + 通话小结)
- **企业版**：定制报价 (私有部署 + 定制模型 + API对接)

## 开发信息

- **开发模式**：一人公司 + AI编程工具辅助开发
- **开发周期**：2-3个月 (业余时间)
