## 项目概览

**急急话术AI** 是一款销售通话实时AI话术辅助桌面工具，主要特点：

- 🎯 **实时话术导航**：客户话音刚落，屏幕即刻弹出推荐话术
- 🧠 **企业知识库即时调用**：从上传文档中自动提取准确答案
- ⚠️ **风险合规预警**：实时监测并警示违规承诺
- 📊 **智能通话小结**：通话结束后自动生成结构化摘要
- 🔒 **本地化部署**：敏感数据不出电脑，隐私安全有保障
- 💰 **超低成本**：99元/月起，无需购买昂贵SaaS系统

## 技术栈

- **应用框架**：Tauri v2 (Rust + React + TypeScript)
- **音频捕获**：cpal (Rust)
- **语音转写**：whisper.cpp
- **向量数据库**：LanceDB
- **Embedding模型**：BGE-M3 (ONNX Runtime)
- **LLM推理**：Qwen2.5-1.5B (GGUF) / 云端API备选

## 开发信息

- **开发模式**：一人公司 + AI编程工具辅助开发
- **开发周期**：2-3个月 (业余时间)
- **开发预算**：≤ 5,000元人民币
- **目标平台**：Windows 10/11 (优先), macOS 12+
- **收入目标**：第一年100万人民币
