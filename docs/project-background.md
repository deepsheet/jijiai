# 文档一：项目背景 (Project Background)

**版本**：v1.0
**更新日期**：2026-03-23
**作者**：一人公司创始人兼AI开发者

## 1. 项目概述

| 项目要素 | 详情 |
|---------|------|
| 产品名称 | 急急话术AI |
| 产品定位 | 销售通话实时 AI 话术辅助桌面工具 |
| 开发模式 | 一人公司 + AI 编程工具辅助开发 |
| 开发环境 | macOS (M 系列芯片) |
| 开发周期 | 2-3 个月 (业余时间) |
| 开发预算 | ≤ 5,000 元人民币 |
| 目标客户 | 销售型团队 (电销、面销、客服) |
| 首发客户 | 合伙人公司 (企业财税和风控咨询公司) |
| 推广渠道 | 社交媒体 + 自媒体 (小红书/抖音/知乎) |
| 收费模式 | 按人头 + 使用量订阅制 |
| 收入目标 | 第一年 100 万人民币 |

## 2. 创始人背景

- **身份**：一人公司创始人，兼任老板、产品经理、AI 开发者
- **技术能力**：熟悉 Web 开发，正在学习 Rust/Tauri 生态
- **开发资源**：1 台 Mac 电脑，AI 编程工具 (Cursor/Windsurf)
- **时间投入**：工作日晚上 + 周末，约 20 小时/周
- **资金限制**：5000 元预算，需精打细算，优先投入核心功能

## 3. 市场机会

根据 2026 年市场调研：

- **销售团队痛点**：75% 的销售无法将客户互动数据转化为可落地的策略 (IDC 数据)
- **现有工具缺陷**：
  - 云端 SaaS 价格高 (年费数千至数万)
  - 本地工具功能单一，无法实时辅助
  - 隐私顾虑：销售数据不愿上传云端
- **市场空白**：低成本、本地化、实时响应的桌面端 AI 话术助手

## 4. 商业模式

### 收费结构

| 套餐 | 价格 | 包含内容 |
|------|------|----------|
| 基础版 | 99 元/人/月 | 实时话术推荐 + 基础知识库 (100MB) |
| 专业版 | 199 元/人/月 | 无限知识库 + 云端备份 + 通话小结 |
| 企业版 | 定制报价 | 私有部署 + 定制模型 + API 对接 |

### 收入测算 (第一年目标 100 万)

- 目标客户数：约 500 人付费
- 平均客单价：166 元/月 (约 2000 元/年)
- 计算：500 人 × 2000 元/年 = 100 万元/年

### 获客路径

- 合伙人公司首发：20 人 (验证产品)
- 自媒体引流转化：200 人
- 口碑转介绍：150 人
- 社交媒体广告：130 人

## 5. 开发策略

### 核心原则

- MVP 优先：2 个月内上线可用版本，不追求完美
- AI 辅助：80% 代码由 AI 生成，人工审核和优化
- 本地优先：默认本地运行，降低服务器成本
- 免费开源：优先使用开源组件，避免授权费用

### 技术选型约束

- 必须支持 Mac 开发、Windows/Mac 双平台发布
- 安装包体积 < 50MB (便于传播)
- 内存占用 < 150MB (兼容低配电脑)
- 无需独立服务器 (降低运维成本)

## 6. 风险与应对

| 风险 | 影响 | 应对策略 |
|------|------|----------|
| 开发进度延迟 | 高 | 严格 MVP 范围，每两周一个可交付版本 |
| 预算超支 | 中 | 优先使用免费额度，API 调用按需开启 |
| 首个客户不满意 | 高 | 与合伙人深度沟通，每周演示迭代 |
| 获客困难 | 高 | 提前运营自媒体，开发期间积累粉丝 |
| 技术兼容性问题 | 中 | 优先保证 Windows 稳定，Mac 版本后续优化 |
